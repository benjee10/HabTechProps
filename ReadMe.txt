HabTechProps 0.1.0
A prop library for Kerbal Space Program 1.8.0+ by Benjee10.

INSTALLATION
Place the contents of GameData/ into your KSP/GameData/ directory and install necessary dependencies.

DEPENDENCIES
This mod is used by Habtech2, SOCK and reDIRECT.
This library should not be installed individually unless you are a mod developer looking to use these props in your own projects.
